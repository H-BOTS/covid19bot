# covid19bot by H-BOTS

# How to Run?
- Install Dependencies
```
#Clone the repository
git clone -m https://github.com/H-BOTS/covid19bot.git

#create Virtual environment
virtualenv covid19bot
covid19bot\Scripts\Activate #windows

# Install dependencies
python upgrade pip
pip3 install rasa
```
## Run commands 

```
rasa init
rasa train
rasa shell
```
